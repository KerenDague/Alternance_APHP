# Rapport d'alternance — AP-HP

> Ce dépôt accompagne mon rapport d'alternance réalisé au sein du pôle **Stratégie Architecture Urbanisation (SAU)** de la **Direction des Services Numériques (DSN)** de l'**Assistance Publique – Hôpitaux de Paris (AP-HP)**, dans le secteur *Référentiels de données santé*.
>
> L'alternance portait sur l'intégration, la maintenance et la distribution des terminologies médicales hébergées dans **ITM**, le serveur multi-terminologies de santé de l'AP-HP.

---

## Contenu du dépôt

Ce dépôt contient les **trois scripts développés** dans le cadre de mes missions, présentés et analysés dans le rapport.


### `export.sh` — Pipeline d'export pour une application consommatrice

### `rdf_to_csv.py` — Extraction des données d'un fichier RDF afin de créer un fichier CSV

### `classification_remboursement.py` — Classification automatique du statut de remboursement de médicaments
